# Proyecto_Sprint7
# Análisis de Vehículos Usados - Aplicación Web

## Descripción del Proyecto
Esta aplicación web interactiva permite explorar y analizar un conjunto de datos de anuncios de venta de coches usados en Estados Unidos. Los usuarios pueden visualizar patrones en los datos y obtener insights sobre el mercado de vehículos usados.

## Funcionalidad
La aplicación proporciona las siguientes características:

- **Histogramas interactivos**: Visualización de la distribución de precios y kilometraje de los vehículos
- **Gráficos de dispersión**: Análisis de la relación entre precio, kilometraje y otras variables como el tipo de vehículo
- **Filtros dinámicos**: Casillas de verificación para personalizar las visualizaciones según las preferencias del usuario
- **Interfaz intuitiva**: Desarrollada con Streamlit para una experiencia de usuario fluida

## Tecnologías utilizadas
- Python
- Streamlit
- Plotly Express
- Pandas

## Cómo ejecutar la aplicación
1. Instalar las dependencias: `pip install -r requirements.txt`
2. Ejecutar la aplicación: `streamlit run app.py`
